Cloud Computing Introduction

클라우드 컴퓨팅 소개

클라우드 특징

  • 주문형 셀프 서비스 - 사용자 스스로 개별 관리
  • 광범위한 네트워크 접속
  • 리소스 공유
  • 신속한 확장성 - scale up/ down
  • 측정 가능한 서비스 - 종량제

클라우드 서비스 모델

  • Iaas(Infrastructure as a Service)
    사업자는 pay-as-you-go access 제공 (ex.storage, networking, servers, ...)
    사진 출처 : 레드햇

  • Paas(Platform as a Service)
    사업자는 cloud-based environment + infrastructure 제공
    사용자는 application 개발
    사진 출처 : 레드햇

  • Saas(Software as a Service)
    사업자는 사용자에게 software/application 제공
    사용자는 subscrible를 하고 web 또는 API를 통해 access
    사진 출처 : 레드햇

장점

  1. 경제성 - 사용하고자 하는 기간만, 소프트웨어/데이터를 클라우드에서 통합 관리
  2. 유연성 - 리소스를 필요할 때, 필요한 만큼 확장/축소 가능
  3. 가용성 - 장애 발생시 계속 사용 가능
  4. 빠른 구축 속도
  5. 손쉬운 글로벌 서비스
  6. 강력한 보안

단점

  1. 생각보다 비싼 비용
  2. 점점 커지는 클라우드 의존성
  3. 데이터 보관의 불안함

주요 클라우드 서비스

  • 가상 서버(virtual server)
    • 로드 밸런스(load balance) - 이중화, 부하 분산
    • 오토 스케일링(auto scaling)
  • 스토리지 서비스(storage)
  • 네트워크 서비스 -VPN, 전용선
  • 데이터베이스

  • 메인페이지로
